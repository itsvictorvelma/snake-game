## How to Play

1. Clone the repository:

```bash
git clone <https://github.com/itsvictorvelma/snake-game>
```
2. Navigate to the project directory:
   
```bash
cd snake-game
```

3. Run the game

```bash
python main.py

```
or 
```bash
python3 main.py
```
4. Use the arrow keys to control the snake and try to eat as much food as possible without hitting the walls or yourself.\
